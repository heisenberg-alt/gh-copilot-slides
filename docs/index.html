PowerPoint

Styled slides with preset colors and fonts. Speaker notes included. Apply branding from existing PPTX templates.

Pixel-perfect render via Playwright (Chromium). One page per slide, print-ready quality.

Get Started
Up and running in minutes.

Choose your interface ‚Äî VS Code Copilot, Go CLI, or Docker.

VS Code Copilot 

$ pip install -e ".[all]"

# Open project in VS Code
# MCP server auto-registers
# Use in Copilot Chat:

@slide-builder Create a presentation
about renewable energy

Go CLI 

$ go build -o bin/slides ./cmd/slides

$ ./bin/slides research \
    --topic "AI in Healthcare" \
    --formats html,pptx,pdf

$ ./bin/slides edit # interactive

Docker 

$ docker build -t slide-builder .

$ docker run \
    -e GITHUB_TOKEN="$GITHUB_TOKEN" \
    slide-builder

Configuration 

# Required (pick one):
GITHUB_TOKEN=ghp_...
OPENAI_API_KEY=sk-...

# Optional:
SLIDE_LLM_MODEL=gpt-5.2
SLIDE_LLM_PROVIDER=copilot
SLIDE_LLM_TIMEOUT=60
